Accessibility & prefers-reduced-motion

‚ö†Ô∏è Important Accessibility Concerns

Animations can cause discomfort for users with vestibular disorders, motion sensitivity, or cognitive disabilities. Some users may experience dizziness, nausea, or headaches when exposed to certain animations.

Always respect user preferences for reduced motion!

Simulate prefers-reduced-motion

Toggle this switch to see how your animation would behave for users who prefer reduced motion.

/* CSS for respecting reduced motion preferences */
@media (prefers-reduced-motion: reduce) {
    * {
        animation-duration: 0.01ms !important;
        animation-iteration-count: 1 !important;
        transition-duration: 0.01ms !important;
        scroll-behavior: auto !important;
    }
}

/* Or more targeted approach */
.animated-element {
    animation: bounce 1s linear infinite;
}

@media (prefers-reduced-motion: reduce) {
    .animated-element {
        animation: none;
        /* Provide alternative visual feedback */
        border: 3px solid #2563eb;
    }
}
